## 1. Description du laboratoire

Ce laboratoire porte sur les mesures de couverture de code avec JaCoCo. Le but principal était d’exécuter les tests existants, d’analyser la couverture obtenue, puis d’ajouter des tests pour améliorer cette couverture.

Le laboratoire demandait aussi de travailler avec la classe `Date`, de réfléchir aux différentes formes de couverture comme la couverture des énoncés, des branches, des conditions et des conditions/branches. Après l’ajout des tests, il fallait aussi refactoriser le code progressivement, tout en s’assurant que les tests continuaient de passer.

D’après les consignes du laboratoire, il fallait aussi créer un `README.md`, ajouter des captures d’écran et référencer clairement le dépôt GitHub pour la soumission.

---

## 2. Objectifs du laboratoire

Les objectifs principaux étaient les suivants :

- exécuter un outil de couverture de code;
- utiliser JaCoCo pour analyser les tests existants;
- ajouter des tests JUnit pour améliorer la couverture;
- appliquer des techniques de couverture en boîte blanche;
- refactoriser le code en petits changements;
- continuer la pratique de Git et GitHub.

Ces objectifs correspondent aux consignes du Lab 03, qui demandent d’exécuter JaCoCo, d’améliorer la couverture de `Date.java`, puis de documenter le travail dans un fichier `README.md`.
